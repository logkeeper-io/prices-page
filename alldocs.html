Generate Token

The Generate Token endpoint is used to authenticate a user or system and return a JSON Web Token (JWT). This token can then be used to access protected resources in the API that require authentication.

Request

Parameter Type Position # Description
Authorization string Header Required The username or email address of the user.
username string Body Required The username or email address of the user.

Response

{
    "status": true,
    "result_code": 200,
    "message": "Success!",
    "values": {
        "name": "Smith",
        "email": "Smith@gmail.com",
        "token": "somaiquahBohngeeshi9woogudofo3eu",
        "account_status": "1"
    }
}
